# Lunar Age & Moon Phase Calendar 🌙📅

A beautiful and practical **Lunar Age Calendar** built with **Flutter**. This application allows users to instantly view the current moon age and explore monthly lunar phases through high-quality, realistic moon imagery.

## 🌟 Overview
This application is designed specifically for the **Japanese market**, offering a serene and intuitive interface for moon enthusiasts and amateur astronomers.

- **Today's Moon Age:** Instantly access the moon's age for the current date.
- **Monthly Overview:** Select any month and year to view a complete grid of daily lunar ages.
- **Realistic Visuals:** Detailed moon images that dynamically change to reflect the specific lunar phase.
- **Navigation:** Easily jump between months to plan lunar observations.

## 🛠 Technical Overview
- **Framework:** Flutter
- **Language:** Dart
- **Logic:** Precise astronomical calculation logic to determine the moon's age based on date and time.
- **Target Market:** Japan 🇯🇵

*Note: This repository contains the core logic and source code. Assets (high-resolution moon images and localization files) are excluded to protect intellectual property.*
